Installation de l'Agent OAM (mode test)

L'Agent OAM est un outil installé sur votre poste qui permet à l'équipe de vous envoyer des actions en temps réel lors des sessions de test (ex. : ouvrir automatiquement une page dans votre navigateur). Il s'exécute en arrière-plan et est visible uniquement par son icône dans la barre des tâches.

Télécharger et installer

Le script télécharge automatiquement l'agent, l'installe dans C:\ProgramData\OAM-Test\ACCEPTATION\, crée un raccourci sur votre bureau et démarre l'agent.

Télécharger l'installateur (PowerShell)

Étapes d'installation

  1. Lancer le script
    Clic droit sur Install-AgentOAM-ACCEPTATION.ps1 → Exécuter avec PowerShell
    Le script s'occupe de tout : téléchargement, extraction et création du raccourci sur le bureau.
  2. Vérifier la présence de l'icône
    Une icône rouge apparaît dans la zone de notification (en bas à droite, près de l'horloge). Si elle n'est pas visible, cliquez sur la flèche ∧ pour afficher les icônes cachées.
    • Point vert — connecté au serveur
    • Point rouge — déconnecté (réseau, VPN, serveur indisponible)

Lancer l'agent en tant qu'un utilisateur de test

Pour simuler la session d'un agent terrain, vous pouvez lancer l'outil sous un compte utilisateur de test (ex. MES\ESAC763) sans ouvrir une session Windows distincte.

  1. Dans l'Explorateur Windows, naviguez jusqu'à C:\ProgramData\OAM-Test\ACCEPTATION\
  2. Maintenez la touche Maj enfoncée, puis faites un clic droit sur l'exécutable
    (ex. Agent.TrayClient.ACCEPTATION.exe)
  3. Dans le menu contextuel, cliquez sur Exécuter en tant qu'autre utilisateur
  4. Dans la fenêtre qui s'ouvre, entrez les informations du compte de test :
    • Nom d'utilisateur : MES\ESAC763
    • Mot de passe : le mot de passe du compte de test
  5. Cliquez sur OK — l'icône de l'agent apparaît dans la barre des tâches sous l'identité de ce compte.

Quitter l'agent

Clic droit sur l'icône dans la barre des tâches → Quitter.
