Télécharger la nouvelle version

Cliquez sur le bouton ci-dessous pour télécharger le fichier ZIP contenant la nouvelle version de l'agent ainsi que sa configuration pour votre environnement.

Télécharger AgentOAM-Side.zip

Étapes d'installation

  1. Fermer l'agent actuel
    Faites un clic droit sur l'icône de l'agent dans la barre des tâches (en bas à droite) et cliquez sur Quitter.
  2. Extraire le ZIP téléchargé
    Clic droit sur AgentOAM-Side.zip → Extraire tout…
    Choisissez un dossier fixe sur votre poste pour le déployer.
  3. Lancer le nouvel agent
    Double-cliquez sur l'exécutable dans le dossier extrait. Son nom contient le nom de votre environnement, par exemple :
    Agent.TrayClient.ACCEPTATION.exe
  4. Vérifier l'icône dans la barre des tâches
    L'agent apparaît avec une icône rouge (mode test). La couleur du point indique l'état de la connexion :
    • Point vert — connecté au serveur
    • Point rouge — déconnecté (réseau, VPN, serveur indisponible)

Première installation ?

Si c'est la première fois que vous installez l'agent, consultez le guide d'installation complet pour les étapes détaillées, les emplacements recommandés et la création d'un raccourci.
