Mise à jour de l'Agent OAM (mode test)

Télécharger la nouvelle version

Téléchargez le script ci-dessous, puis lancez-le. Il arrête l'agent en cours, remplace les fichiers, met à jour le raccourci sur votre bureau et redémarre automatiquement.

Télécharger l'installateur (PowerShell)

Étapes

  1. Clic droit sur Install-AgentOAM-ACCEPTATION.ps1 → Exécuter avec PowerShell
  2. Le script s'occupe du reste — lancez ensuite l'agent via le raccourci sur votre bureau.

Première installation ?

Si c'est la première fois que vous installez l'agent, consultez le guide d'installation complet pour les étapes détaillées, les emplacements recommandés et la création d'un raccourci.
