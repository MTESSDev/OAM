Installation de l'Agent OAM (mode test)

L'Agent OAM est un outil installé sur votre poste qui permet à l'équipe de vous envoyer des actions en temps réel lors des sessions de test (ex. : ouvrir automatiquement une page dans votre navigateur). Il s'exécute en arrière-plan et est visible uniquement par son icône dans la barre des tâches.

Télécharger l'agent

Le fichier ZIP contient l'exécutable de l'agent ainsi que sa configuration pour votre environnement de test.

Télécharger AgentOAM-Side.zip

Étapes d'installation

  1. Extraire le ZIP
    Clic droit sur AgentOAM-Side.zip → Extraire tout…
    Extrayez dans C:\ProgramData\OAM-Side\
    ProgramData est un dossier système partagé entre tous les comptes Windows. Il peut être caché dans l'Explorateur — copiez le chemin directement dans la barre d'adresse et appuyez sur Entrée.
  2. Lancer l'agent
    Dans le dossier extrait, double-cliquez sur l'exécutable. Son nom contient le nom de votre environnement, par exemple :
    Agent.TrayClient.ACCEPTATION.exe
  3. Vérifier la présence de l'icône
    Une icône apparaît dans la zone de notification (en bas à droite de l'écran, près de l'horloge). Si elle n'est pas visible, cliquez sur la flèche ∧ pour afficher les icônes cachées.
    • Point vert — connecté au serveur
    • Point rouge — déconnecté (réseau, VPN, serveur indisponible)

Créer un raccourci

Pour retrouver facilement l'agent ou le relancer sans naviguer dans l'Explorateur :

  1. Naviguez jusqu'à C:\ProgramData\OAM-Side\ dans l'Explorateur Windows
  2. Clic droit sur l'exécutable (ex. Agent.TrayClient.ACCEPTATION.exe) → Envoyer vers → Bureau (créer un raccourci)

Lancer l'agent en tant qu'un utilisateur de test

Pour simuler la session d'un agent terrain, vous pouvez lancer l'outil sous un compte utilisateur de test (ex. MES\ESAC763) sans ouvrir une session Windows distincte.

  1. Dans l'Explorateur Windows, naviguez jusqu'au dossier d'extraction choisi
  2. Maintenez la touche Maj enfoncée, puis faites un clic droit sur l'exécutable
    (ex. Agent.TrayClient.ACCEPTATION.exe)
  3. Dans le menu contextuel, cliquez sur Exécuter en tant qu'autre utilisateur
  4. Dans la fenêtre qui s'ouvre, entrez les informations du compte de test :
    • Nom d'utilisateur : MES\ESAC763
    • Mot de passe : le mot de passe du compte de test
  5. Cliquez sur OK — l'icône de l'agent apparaît dans la barre des tâches sous l'identité de ce compte.

Quitter l'agent

Clic droit sur l'icône dans la barre des tâches → Quitter.
